OMA groups contain sets of genes which are all orthologous to one another within the group. This implies that there is at most one entry from each species in a group.

OMA Group IDs

  • Fingerprint
  • Group Nr

The fingerprint is a subsequence that is specific to a certain group in the current OMA release. Since the group number is specific to a release, this is the best way to track a group across releases.

The numeric OMA Group id is release specific. It is therefor not recommended to use it to track a group between different releases. Use instead the fingerprint.
